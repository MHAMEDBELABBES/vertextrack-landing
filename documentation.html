Vue d'ensemble

Pipeline de traitement

VERTEXTRACK implémente un pipeline end-to-end qui transforme une vidéo frontale en un score de risque LCA accompagné d'explications SHAP par feature biomécanique. Chaque étape est conçue pour être reproductible et non-invasive.

01

Acquisition vidéo

Vidéo 1080p à 60fps. Tests supportés : CMJ (Counter-Movement Jump), SJ (Squat Jump), DVJ (Drop Vertical Jump). Positionnement caméra standardisé à 2m de distance.

60fps · 1080p
02

Extraction de pose — MediaPipe BlazePose

33 landmarks 3D extraits frame-par-frame. Coordonnées normalisées par rapport à la largeur des hanches. Confiance minimale de 0.5 requise pour valider un landmark.

33 landmarks
03

Filtrage Savitzky-Golay

Lissage des trajectoires articulaires avec fenêtre de 11 frames et polynôme d'ordre 3. Élimine le bruit de capture tout en préservant les pics de vélocité angulaire critiques.

window=11 · poly=3
04

Calcul des features biomécaniques

12 features extraites par session : FPPA bilatéral, angles de flexion genou/hanche, LSI (Limb Symmetry Index), vélocités angulaires, hauteur CMJ par méthode temps de vol et déplacement CoM.

12 features
05

Prédiction Random Forest + SHAP

Modèle Random Forest (100 estimateurs) entraîné en LOSO-CV. Score de probabilité de risque LCA entre 0 et 1. Valeurs SHAP calculées par TreeExplainer pour chaque prédiction.

AUC-ROC 0.80
Extraction de pose

MediaPipe BlazePose

BlazePose est un modèle de détection de pose développé par Google MediaPipe. Il détecte 33 landmarks squelettiques en temps réel, incluant les articulations clés pour l'analyse biomécanique du membre inférieur.

import mediapipe as mp import cv2 mp_pose = mp.solutions.pose pose = mp_pose.Pose( static_image_mode=False, model_complexity=2, # modèle haute précision min_detection_confidence=0.5, min_tracking_confidence=0.5 ) # Extraction landmarks par frame results = pose.process(cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)) landmarks = results.pose_landmarks.landmark

Les landmarks utilisés pour l'analyse LCA : LEFT_HIP (23), RIGHT_HIP (24), LEFT_KNEE (25), RIGHT_KNEE (26), LEFT_ANKLE (27), RIGHT_ANKLE (28), LEFT_HEEL (29), RIGHT_HEEL (30).

Modèle machine learning

Random Forest — LOSO-CV

Le modèle de classification est un Random Forest scikit-learn entraîné avec la stratégie Leave-One-Subject-Out Cross-Validation (LOSO-CV), qui garantit une évaluation non biaisée en excluant toutes les sessions d'un même athlète du set d'entraînement à chaque fold.

from sklearn.ensemble import RandomForestClassifier from sklearn.model_selection import LeaveOneGroupOut clf = RandomForestClassifier( n_estimators=100, max_depth=None, min_samples_split=2, random_state=42, class_weight='balanced' # compense le déséquilibre de classes ) logo = LeaveOneGroupOut() # groups = identifiant athlète (garantit isolation inter-sujet) scores = cross_val_score(clf, X, y, groups=groups, cv=logo, scoring='roc_auc')
Explicabilité

SHAP — SHapley Additive exPlanations

Chaque prédiction est accompagnée de valeurs SHAP calculées par TreeExplainer, qui décomposent le score de risque en contributions individuelles par feature. Cette explicabilité est essentielle pour que les entraîneurs et kinés puissent cibler les déficits biomécaniques spécifiques à corriger.

import shap explainer = shap.TreeExplainer(clf) shap_values = explainer.shap_values(X_athlete) # shap_values[1] = contributions pour la classe "risque élevé" # Feature la plus impactante : fppa_left (valgus dynamique) shap.summary_plot(shap_values[1], X_athlete, feature_names=feature_cols)

Un FPPA élevé (valgus dynamique du genou) est systématiquement la feature la plus contributive au score de risque dans les profils à risque élevé identifiés sur le dataset USB + RCOZ.

Protocoles de test

Trois tests biomécanique

⬆
CMJ
Counter-Movement Jump
Saut vertical avec contre-mouvement depuis position bipodale. Vue sagittale ou frontale. Mesure la puissance explosive des membres inférieurs.
→ Hauteur de saut (temps de vol)
→ Flexion genou au décollage
→ Symétrie bilatérale
⬇
SJ
Squat Jump
Saut vertical depuis position squat statique (sans contre-mouvement). Vues sagittale + frontale. Isole la composante concentrique pure et détecte les déficits de symétrie bilatérale.
→ Hauteur de saut SJ (cm)
→ ROM flexion genou au décollage
→ Valgus dynamique (vue frontale)
↓
DVJ
Drop Vertical Jump
Chute depuis une boîte de 30cm suivie d'un saut vertical maximal. Vue frontale. Test le plus discriminant pour l'évaluation du risque LCA à l'atterrissage.
→ Valgus excursion à l'atterrissage
→ Vélocité angulaire genou
→ FPPA au contact initial
